Creating a project

If you have an admin token you can create a project. To create a project,

  • Choose the appropriate circle for the project.
  • Fill a short name/description of the project.
  • Fill an expiration date in YYYY-MM-DD format. For example, 2024-02-29
  • Decide the amount of ADA and amount of token (gimbals) that is appropriate as incentive/compensation for the project.
  • Additionally you may create a page in notion upfront and put that link to keep track of the details of the project.

Committing to a project

If you have an contributor token you can commit to a project.

Note: Before you commit to a project you have to make sure if your reward address is configured to your current address where your contributor token is held.
